### Important Note on File Structure for the Rolex Program 
The correct placement of files in subdirectories is crucial. If an image file is not found in its expected location, the program will hang with a white window and no error message, making it difficult to diagnose the problem. 
Please be careful when changing directories!

1. **Rolex.pde** - This example demonstrates how to construct forms (needles) that update every second.
2. **Demo_rotate.pde** - This example uses single images for the needles and rotates them using the `rotate` method.

### Expected Directory Structure
By organizing the files according to this structure, you can avoid issues and ensure the smooth functioning of the program.
